# Covid19 Global Tracker

> [View Demo](https://global-covid19-tracker.web.app/)
>
> This is a simple covid19 global tracker using https://covid19.mathdro.id/api/

## Technologies

> 1. `React Hooks`
> 2. `Material UI`
> 3. `react-chartjs-2`

## Instructions:

> 1. `Clone the Repo`
>
> 2. `npm install`
>
> 3. `npm start`
